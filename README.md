# LLM Experiment Lab

GUI-приложение для экспериментов с LLM моделями.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Настройка

При первом запуске приложение автоматически создаст конфигурационный файл в домашнем каталоге пользователя: `~/.config/llmexplab/config.json`.

Если у вас уже есть файл `config.json` в директории с программой, он будет автоматически перенесён в профиль пользователя.

## Хранение данных

- Конфигурация: `~/.config/llmexplab/config.json`
- Эксперименты: `~/.config/llmexplab/experiments/`

Каждый эксперимент сохраняется в отдельном подкаталоге с именем `YYYY-MM-DD_HH-MM-SS_название/` и содержит:
- `experiment.json` - параметры и результаты эксперимента
- `notes.md` - заметки в формате Markdown
